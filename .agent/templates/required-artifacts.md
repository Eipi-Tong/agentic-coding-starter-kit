# Required Repository Artifacts

Every project created with this starter kit must contain the following artifacts.

---

# 1. Project Summary

File:

docs/architecture/project-summary.md

Contains:

- Product overview
- Target users
- Core features
- Tech stack
- System architecture summary

Purpose:
Allows agents and developers to quickly understand the system.

---

# 2. Bootstrap Form

File:

.agent/templates/bootstrap-form.md

Purpose:

Defines the project configuration and initial decisions.

---

# 3. Agent Rules

Directory:

.agent/rules/

Required files:

decision-boundaries.md  
definition-of-done.md

Purpose:

Defines agent governance and coding standards.

---

# 4. Feature Request Template

File:

.agent/templates/feature-request.md

Purpose:

Standard format for requesting new functionality.

---

# 5. Bugfix Template

File:

.agent/templates/bugfix-request.md

Purpose:

Standard format for reporting and fixing bugs.

---

# 6. Refactor Template

File:

.agent/templates/refactor-request.md

Purpose:

Standard format for code restructuring tasks.

---

# 7. README

File:

README.md

Contains:

- Project purpose
- Setup instructions
- Development workflow
- Links to documentation
